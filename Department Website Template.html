3 Easy Ways to Put Your Site on the Web

1. GitHub Pages (Free & Reliable)

Perfect if you’re comfortable with a bit of basic Git and file upload.

Steps:

  1. Create a GitHub account (if you don’t have one): github.com

  2. Create a new repository on GitHub (call it e.g. department-website).

  3. On your computer, save the HTML file you created as index.html.

  4. Upload your index.html file and any images (in /images/ folder) to the repository.

    • You can drag and drop files directly on GitHub in the repo’s "Add file" → "Upload files" section.

  5. Go to Settings > Pages (on your GitHub repo).

  6. Under Source, select the branch (usually main or master) and root folder (/).

  7. Save — GitHub will give you a website link like https://yourusername.github.io/department-website/

  8. Visit that link and your site will be live!

2. Netlify Drop (Super Easy for Beginners)

No accounts or command lines needed!

Steps:

  1. Go to Netlify Drop

  2. Drag and drop your entire website folder (containing index.html and the /images/ folder).

  3. Wait a few seconds while it uploads.

  4. Netlify will give you a random URL you can share instantly.

  5. You can customize the URL later if you sign up for free.

3. Google Drive or Dropbox (Limited)

Not ideal for a full website, but quick for static HTML and testing.

What You Need to Prepare Before Uploading

  • Your index.html file.

  • An /images/ folder with all your photos used in the gallery.

  • Make sure image links in your HTML are relative, e.g. <img src="images/photo1.jpg">.

How to Put Your Website Online (Step-by-Step)

Step 1: Create a GitHub Account

  • Go to github.com

  • Click Sign up and create a free account.

Step 2: Create a New Repository (Your Website Folder)

  • After logging in, click the + icon (top right), then New repository.

  • Name it something like department-website.

  • Leave it Public.

  • Click Create repository (don’t check anything else).

Step 3: Upload Your Website Files

  • On the new repository page, click Add file → Upload files.

  • Upload your index.html file.

  • Create a folder called images on your computer, put your photos there, then upload the whole folder (you can upload multiple files by dragging them in or clicking “choose your files”).

  • Scroll down and click Commit changes to finish uploading.

Step 4: Enable GitHub Pages to Publish Your Site

  • In the repo, click Settings (tab near top).

  • Scroll down to Pages on the left menu.

  • Under Source, select the branch main (or master) and folder / (root).

  • Click Save.

Step 5: Open Your Live Website!

  • After a few seconds, GitHub will show your website link under GitHub Pages (usually looks like https://yourusername.github.io/department-website/).

  • Click the link to see your site live on the web!
